# Dataset Source

Save this file as `data/SOURCE.md` in your repo.

The dataset used by this tool is **not bundled in this repository**. It is
published separately and should be downloaded from:

**Figshare:** https://figshare.com/s/7da6474b776654003959

**Citation:**
Anik, F. W., Zaman, M., Foyez, T., & Mamun, K. A. *DOSAGE: A Dataset for
Optimized Safe Antibiotic Guidelines and Estimates*, Nature Scientific Data,
2025. DOI: https://www.doi.org/10.1038/s41597-025-05761-8

After downloading, place the CSV files in this `data/` folder — see
`data/README.md` for the exact filenames and column mapping this app expects.
